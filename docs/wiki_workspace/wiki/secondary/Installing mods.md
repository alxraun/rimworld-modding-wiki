## Installing Mods - Cheat Sheet

**Using Mods:**

- **Steam Workshop (Steam version):**
    - Easiest way to find, download, and update mods.
    - Automated installation and updates.

- **Manual Installation (All versions):**
    - Download mods (Steam Workshop, Nexus Mods, GitHub, etc.).
    - Unzip mod folders into the local `Mods` folder.

**Manual Installation - Folder Locations by OS:**

- **Windows:** `C:\Program Files (x86)\Steam\steamapps\common\RimWorld\Mods`
- **Mac:** `~/Library/Application Support/Steam/steamapps/common/RimWorld/RimWorldMac.app/Mods`
- **Linux (standalone):** `~/.steam/steam/steamapps/common/RimWorld/Mods`
- **Linux (GoG):** `/home/<user>/GOG/Games/RimWorld/game/Mods/`

**Activating Mods:**

1. Install mods (Steam Workshop or Manual).
2. Launch RimWorld.
3. Select "Mods" from the main menu.
4. Activate/deactivate mods by toggling checkboxes.
5. Restart RimWorld for changes to take effect.

**Making Mods:**

- Refer to the [Modding Tutorials](https://rimworldwiki.com/wiki/Modding_Tutorials) hub for guides and resources.
