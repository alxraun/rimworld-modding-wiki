## Modding - Cheat Sheet

**What are Mods?**

- Game modifications altering RimWorld content and functionality (add, change, remove).
- RimWorld supports mods, including Steam Workshop.

**Using Mods:**

- **Steam Workshop (Steam version):**
    - Easiest way to find, download, and update mods.
    - Automated installation and updates.

- **Manual Installation:**
    - For non-Steam or specific mods.
    - Install Location: `RimWorld/Mods/YourModName`.
        - **Windows:** `C:\Program Files (x86)\Steam\steamapps\common\RimWorld\Mods`
        - **Mac:** `~/Library/Application Support/Steam/steamapps/common/RimWorld/RimWorldMac.app/Mods`
        - **Linux (standalone):** `~/.steam/steam/steamapps/common/RimWorld/Mods`
        - **Linux (GoG):** `/home/<user>/GOG/Games/RimWorld/game/Mods/`
    - Installation Steps: Unzip mod into subfolders within `Mods` folder.

- **Activating Mods:**
    - Via "Mods" option in RimWorld main menu.
    - Requires game restart to activate/deactivate.

**Making Mods:**

- Refer to the "[Modding Tutorials](https://rimworldwiki.com/wiki/Modding_Tutorials)" hub for guides and tutorials.
